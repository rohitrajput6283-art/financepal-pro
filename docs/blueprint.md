# **App Name**: FinancePal Pro

## Core Features:

- EMI Calculator: Calculate Equated Monthly Installments with adjustable sliders for principal, interest rate, and tenure, displaying total interest and total payment.
- SIP Calculator: Calculate the maturity amount for a Systematic Investment Plan based on user inputs.
- GST Calculator: Compute Goods and Services Tax (GST) for specified rates: 5%, 12%, 18%, and 28%.
- Budget Tracker: Allow users to input and manage income and expense entries, persisting data in a backend database.
- AI Financial Assistant tool: Provide personalized spending tips based on tracked expenses or clarify complex financial terms relevant to the calculations.
- Currency Converter: Convert currencies using live exchange rates fetched from an external API.
- User Authentication: Enable secure user login via Firebase, supporting email/password and Google authentication.

## Style Guidelines:

- Primary color: Professional and trustworthy blue (#147AEA). Hue is 210, with good saturation for a modern finance application.
- Background color: A very light, subtle blue (#E8EDF2). Shares the primary hue (210) but is highly desaturated (20%) and bright, offering a clean, professional canvas for the light theme.
- Accent color: Vibrant and growth-oriented green (#2EB34E). Selected for contrast and to align with the 'blue and green Material Design' request, representing financial growth. Its hue is sufficiently different from the primary blue, and it's set at a higher saturation for impact.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective appearance, ensuring excellent readability for numerical and textual financial information, consistent with Material Design principles.
- Utilize a consistent set of vector-based financial and calculator icons, adhering to Material Design 3 guidelines for a clean and intuitive user experience.
- Implement a tab-based navigation system at the bottom for easy access to main features. Utilize clean, card-like structures for each calculator and budget entry to segment content visually.
- Incorporate subtle transitions and micro-interactions, such as smooth slider adjustments and tab changes, to enhance the app's responsiveness and overall user experience, in line with Material Design 3.
